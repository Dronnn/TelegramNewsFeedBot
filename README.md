# Telegram News Feed Bot

Бот для агрегации новостей из Telegram-каналов в единую персональную ленту.

## Возможности

- **Ручная подписка на каналы** — добавляйте любые публичные Telegram-каналы и получайте их посты в одном месте.
- **Тематическая лента** — подписывайтесь на темы (~30 категорий: технологии, наука, спорт и др.), бот автоматически подберёт каналы.

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Язык | Python 3.11+ |
| Bot API | aiogram 3.x |
| Userbot / парсинг каналов | Telethon |
| База данных | SQLite + aiosqlite |
| Конфигурация | python-dotenv |

## Установка и запуск

1. Клонировать репозиторий:
   ```bash
   git clone <repo-url>
   cd TelegramNewsFeedBot
   ```

2. Создать виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Скопировать `.env.example` в `.env` и заполнить реальными значениями:
   ```bash
   cp .env.example .env
   ```

5. Запустить бота:
   ```bash
   python -m bot.main
   ```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и регистрация пользователя |
| `/add <канал>` | Подписаться на канал |
| `/remove <канал>` | Отписаться от канала |
| `/list` | Список текущих подписок |
| `/topics` | Показать доступные темы |
| `/mytopics` | Управление подписками на темы |
| `/pause` | Приостановить ленту |
| `/resume` | Возобновить ленту |
| `/help` | Справка по командам |
